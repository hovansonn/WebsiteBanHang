# WebsiteBanHang

Website bán hàng có một số tính năng như:
  - Hiển thị các mặt hàng, top các mặt hàng bán chạy, top các mặt hàng mới nhất.
  - Cho phép người dùng tìm kiếm sản phẩm theo nhiều tiêu chí như: Tên sản phẩm, loại sản phẩm, nhà sản xuất, ...
  - Mua hàng, đặt sản phẩm vào giỏ hàng, thanh toán giỏ hàng.
  - Đăng nhập, đăng xuất.
  - Trang admin quản lí các hàng hoá và người dùng.
### Môi trường
  1. Front-end : HTML5, CSS, JavaScript, Jquery, Bootstrap.
  2. Back-end : Thuần PHP.
  3. Database : MySQL.
### Cài Đặt
  1. Cài đặt XAMPP hoặc WAMPP (đối với OS là Window), LAMP (đối với OS là Linux), MAMPRO (đối với OS là MAC) 
  2. Chạy file qllaptop.sql trong thư mục database với lên SQL server trong máy
  3. Copy thư mục chứa code vào thư mục htdocs trong thư mục xampp
  4. Chạy web trên địa chỉ localhost
